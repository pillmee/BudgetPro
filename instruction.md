# 구현
- 파이썬과 fastAPI를 사용하여 구현한다.
- 앱이름은 BudgetPro 이다.

# 기능

- 다중 팀을 추가, 수정, 삭제할 수 있다.
- 팀원을 추가, 수정, 삭제할 수 있다.
- 팀, 팀원을 선택하여 로그인할 수 있다.
- 예산은 매월 “인별 할당액 x 팀원 수” 만큼씩 누적된다.
- 전체 주기는 1월부터 12월까지이다. 예산은 1월부터 누적되기 시작한다.
- 지출액을 입력하면 공급가액과 부가세 10%를 자동으로 계산하여 입력한다.
- 지출액을 입력할 때 카테고리는 “회식, 음료, 사무용품, 기타”로 지정할 수 있고, 지출 내용을 적을 수 있다.
- 잔여 예산은 설정된 주기에 따라 초기화가 이뤄진다. 주기는 월별, 분기별, 반기별, 연도별로 설정할 수 있다.
- 전체주기의 종료월인 12월이 되면 예산이 초기화된다.
- 팀원, 초기화 주기, 인별 금액등은 json 파일로 팀단위로 관리된다.
- 지출정보는 별도의 json 파일로 관리한다.

# 화면

## 시작화면

- 팀을 선택할 수 있다.
- 팀을 추가할 수 있다.

## 로그인 화면

- 팀에 추가된 팀원 중 하나를 선택하여 로그인 할 수 있다.

## 메인 화면

- 로그아웃 하고 팀 선택 화면으로 이동할 수 있다,

### 대시보드

- 누적 예산, 현재 지출, 잔여예산, 잔여예산(부가세 제외) 를 나타낸다.
- 현재 지출은 공급 가액과 부가세를 따로 표시한다.
- 현재 월과 예산 사이클 정보를 나타낸다.
- 팀원들의 사용 내역을 간략하게 그래프로 나타낸다.

### 지출관리

- 지출 내역을 입력할 수 있다.
- 모든 팀원의 지출 내역을 리스트로 나타낸다.

### 팀원 관리

- 팀원을 등록하고, 삭제, 수정할 수 있다.

### 설정

- 지출정보를 json 파일로 다운로드 할 수 있다.
- json 파일을 업로드하여 지출정보를 복구 할 수 있다.
- 지출 정보를 초기화 할 수 있다.
- 인별 금액을 설정할 수 있다.
- 초기화 주기를 설정할 수 있다.
